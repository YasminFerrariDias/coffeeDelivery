# Coffee Delivery

Aplicação de e-commerce de cafés com carrinho de compras, formulário de endereço e confirmação de pedido, desenvolvida com React e TypeScript.

## Tecnologias

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- [React Router DOM](https://reactrouter.com/) — navegação entre páginas
- [Context API](https://react.dev/reference/react/createContext) — gerenciamento de estado global do carrinho

## Funcionalidades

- Listagem de cafés disponíveis
- Adição e remoção de itens no carrinho
- Controle de quantidade por item
- Formulário de endereço de entrega
- Seleção de método de pagamento
- Página de confirmação do pedido

## Como Executar

```bash
npm install
npm run dev
```
